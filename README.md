# 🩺 CareOrbit

CareOrbit is an AI-assisted healthcare management platform designed to connect patients, doctors, and hospital operations through a unified web system.

The project focuses on improving healthcare workflow by providing patient services, symptom analysis, appointment management, medical records, and hospital dashboard features.

## 🚀 Features

### Patient Module
- Patient registration and login
- Patient dashboard
- Symptom analysis system
- Appointment booking
- Medical reports access
- Health recommendations

### Doctor Module
- Doctor authentication
- Doctor dashboard
- Patient records view
- Appointment management
- Prescription management

### Hospital Management Module
- Hospital operations dashboard
- Ward and bed management
- Pharmacy monitoring
- Laboratory overview
- Ambulance management

## 🛠️ Technologies Used

Frontend:
- HTML
- CSS
- Bootstrap
- JavaScript

Backend:
- Python
- Flask

Database:
- SQLite
- Flask-SQLAlchemy
